# 酷我点歌使用说明

本版本已把点歌默认接入配置的酷我 API。

## 命令

```text
!search 稻香
!play 1
!play 稻香
```

- `!search 歌名`：搜索并显示歌曲列表。
- `!play 数字`：播放你最近一次搜索结果中的对应歌曲，序号从 `1` 开始。
- `!play 歌名`：直接搜索该歌曲并播放第一条结果。

播放时，机器人会使用 API 返回的封面作为当前歌曲封面；若 Bot 已启用播放头像更新，封面也会用于机器人头像。

本定制版不再支持用 `!play` 直接播放普通网页或音频链接。运行前仍需要按 TS3AudioBot 原有要求配置 `ffmpeg`。
