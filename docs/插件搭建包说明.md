# 酷我插件搭建包说明

本搭建包使用官方 TS3AudioBot 加独立插件，不修改 Bot 的音乐解析代码。

插件位于 `plugins/KuwoMusicPlugin.dll`。搭建包已将官方 `!play` 的有参数行为改为酷我点歌：

```text
!play 稻香
!search 稻香
!play 0
```

首次启动并完成机器人连接后，先向机器人发送一次：

```text
!plugin load KuwoMusicPlugin.dll
```

`!search 歌名` 会显示酷我搜索结果；`!play 歌名` 会搜索酷我并播放第一首；`!play 数字` 会播放你上次搜索结果中的对应歌曲，编号从 `0` 开始。普通网址播放不再由 `!play` 支持。播放时，Bot 的简介会显示当前歌曲标题；停止播放后会显示“休眠中”。请在 Bot 配置中保持 `set_status_description = true`，并给机器人 TS3 身份授予修改自身简介的权限。

Windows 搭建包已预置 `ffmpeg.exe`。Linux 请安装：

```bash
sudo apt update
sudo apt install -y ffmpeg libopus0 libopus-dev
```

插件 DLL 必须与搭建包中的 TS3AudioBot 版本配套。升级 Bot 时，请重新编译插件或继续使用配套版本。
